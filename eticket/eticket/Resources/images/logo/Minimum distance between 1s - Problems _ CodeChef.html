Problem

Chef has a binary string SSS of size NNN. Chef can perform the following operation on the string:

  • Select any substring of size 333 and reverse it.

Find the minimum distance between any two 111s Chef can achieve, by applying the above operation any (possibly zero) number of times.

Note:

  • The distance between two characters SiS_iSi and SjS_jSj is defined as abs(ij)abs(i-j)abs(ij).
  • It is guaranteed that the string contains at least two occurrences of 111.
  • A substring is obtained by deleting some (possibly zero) elements from the beginning and some (possibly zero) elements from the end of the string.

Input Format

  • The first line of input will contain a single integer TTT, denoting the number of test cases.
  • Each test case consists of multiple lines of input.
    • The first line of each test case contains an integer NNN the number of characters in the string.
    • The next line contains a binary string of length NNN.

Output Format

For each test case, output on a new line, the minimum distance between any two 111s Chef can achieve, by applying the above operation any (possibly zero) number of times.

Constraints

  • 1T10001 \leq T \leq 10001T1000
  • 2N1052 \leq N \leq 10^52N105
  • SiS_iSi contains 000 and 111 only.
  • It is guaranteed that SSS contains at least two occurrences of 111.
  • The sum of NNN over all test cases won't exceed 21052\cdot 10^52105.

Output

Explanation:

Test case 111: There are only two 111s in the string. The distance between the 111s is 21=12-1 = 121=1.

Test case 222: The distance between the given 111s is 31=23-1 = 231=2. It can be shown that we cannot achieve a distance smaller than 222 by applying any number of operations.

Test case 333: Chef can perform the following operation:

  • Select the substring S[2,4]=100S[2,4] = 100S[2,4]=100 and reverse it. Thus, the string becomes S=0001101S = 0001101S=0001101. The minimum distance between the 111s in the string is 54=15-4 = 154=1.

It can be shown that we cannot achieve a distance smaller than 111 by applying any number of operations.
